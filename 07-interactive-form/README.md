# 07 — Interactive Form

**Assignment Date:** 02/03/2026
**Assignment:** Build a form that validates user input and shows success/error messages.

---

## What I Built

A signup form that validates all fields on the client side using JavaScript — shows inline error messages for invalid inputs and a success message when the form is correctly filled.

---

## Features

- Validates Full Name, Email, Password, Role (dropdown), and Terms checkbox
- Inline error messages shown below each field
- Red border highlight on invalid fields
- Success message shown on valid submission
- Fully responsive — works on mobile and desktop

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla / DOM Manipulation)

---

## Project Structure

```
07-interactive-form/
│
├── index.html      # Signup form structure
├── styles.css      # Form styling, error and success states
└── script.js       # Validation logic
```

---

## How to Run

1. Open `index.html` in any browser
2. Try submitting with empty or invalid fields to see error messages
3. Fill everything correctly to see the success message

---

## What I Learned

- How to select and manipulate DOM elements with JavaScript
- How to validate different input types (text, email, password, select, checkbox)
- How to dynamically show/hide error and success messages
- How to add/remove CSS classes with JavaScript for visual feedback
