# 04 — Responsive Hero Page

**Assignment Date:** 21/02/2026
**Assignment:** Build a responsive landing section using Flexbox that adjusts for mobile.

---

## What I Built

A fully responsive landing page with a navbar, hero section, features section, contact form, and footer — built using HTML and CSS with Flexbox for layout and media queries for responsiveness.

---

## Features

- Responsive navbar with a hamburger menu (CSS-only, no JavaScript)
- Hero section with headline, subtitle, CTA buttons, and an image
- Features section with three cards
- Contact form section
- Footer
- Fully responsive for mobile, tablet, and desktop

---

## Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)

---
