## What I Built

A React-based product listing UI that displays items in a grid layout with category filters. Users can switch between categories like Electronics, Fashion, Kitchen, and Food to view relevant products.

---

## Features

* Product cards with image, name, price, and category
* Filter buttons (All / Electronics / Fashion / Kitchen / Food)
* Dynamic filtering using React state
* Responsive grid layout using CSS Grid
* Clean UI with hover effects
* Reusable ProductCard component

---

## Technologies Used

* React
* useState
* CSS Grid
