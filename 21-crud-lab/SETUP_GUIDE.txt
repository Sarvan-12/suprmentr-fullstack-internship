SETUP & RUNNING GUIDE FOR CRUD LAB
==================================

PREREQUISITES:
- Node.js installed
- MongoDB running locally on port 27017
- Thunder Client installed (VS Code extension)

---

STEP 1: INSTALL DEPENDENCIES
cd 21-crud-lab
npm install

Wait for it to finish. You should see node_modules folder created.

---

STEP 2: START MONGODB
Make sure MongoDB is running. In a separate CMD window:
mongod

Or use MongoDB Compass to verify it's running.

---

STEP 3: START THE SERVER
In the 21-crud-lab folder:
npm start

You should see:
MongoDB Connected: localhost
Server running on http://localhost:5000

---

STEP 4: OPEN THUNDER CLIENT
In VS Code:
- Extensions (Ctrl+Shift+X)
- Search "Thunder Client"
- Install it
- Click Thunder Client icon on left sidebar

---

STEP 5: TEST THE API
Follow the requests in THUNDER_CLIENT_REQUESTS.txt

Copy-paste each request one by one.

---

STEP 6: TAKE SCREENSHOTS
For each successful request:
- Right-click → Screenshot
- Save in screenshots/ folder
- Name them: 01-create-product, 02-get-all, etc.

---

STEP 7: UPDATE README
Follow the README template and add:
- What you built
- Features
- The CRUD operations tested
- Screenshots

---

TROUBLESHOOTING:

Q: MongoDB connection error
A: Make sure mongod is running in another terminal

Q: Port 5000 already in use
A: Change PORT in server.js to 3000 or 5001

Q: npm install fails
A: Delete package-lock.json and node_modules, then npm install again
